// TRUST NETWORK — LIVE SIMULATION
▪ 登録済み ▪ 招待待ち ─── 信頼の連鎖

招待コードは信頼できる人だけに手渡し（DM）で渡されます。
コードを持っているということは、誰かに信頼されているということ。
それがこのネットワークの唯一の入場資格です。

💡 コードはDMで一対一で手渡されます。SNSで公開されたコードは無効化されます。信頼している相手にだけ渡してください。「誰を繋ぐか。それは、誰と運命を共にするかという問いだ。」

// 02 — CHAIN

信頼はバトンのように
連鎖していく

コードを受け取って登録すると、あなたにも招待コードが3枚届きます。
信頼できる株クラ仲間に渡して、チェーンをつなげてください。

登録した人は緑のノードとして表示されます。
推薦されたけどまだ未登録の人は青のノード。
招待コードが届いていつでも緑に変わることができます。

// 03 — TRUST SCORE

矢印が集まるほど
信頼度が上がる

同じアカウントを複数の人が推薦すると、ネットワーク図に
複数の矢印が集中します。これが「信頼度」の正体です。

フォロワー数でも、バッジでも、自己申告でもない。
信頼している人たちが実名で指名した回数だけが信頼度になる。
それが株クラ X Network の評価軸です。

// 04 — REAL NUMBERS

上限1,000でも
実際の参加人数はもっと少ない

AさんとBさんが同じ@Cさんを推薦した場合、
Cさんが使うコードは1枚だけ。もう1枚は余る。

つまり人気アカウントに矢印が集まるほど、コードの消費が減り、
上限1,000よりさらに少ない人数でネットワークが完成する。
段理論上限現実予測（消化率50%）
1段目最初の招待組〜35人
2段目×3〜80人
3段目×3〜150人
4段目×3〜200人
合計最大1,000人実質 400〜600人

400〜600人という規模は信頼の密度が最も高いサイズ感です。
大きすぎず、薄まらず。本物だけが集まる濃いコミュニティ。

// 05 — HOW TO JOIN

参加までの流れ

01
招待コードを受け取る
すでにネットワークに参加している株クラ仲間から
招待コード（KBK-XXXX形式）をDMで受け取る
02
ハンドルと推薦を登録
招待コード・あなたの@ハンドル・
信頼できる株クラ仲間を1〜3人入力して登録
03
あなたの招待コードが3枚届く
登録完了と同時に次の招待コードが3枚発行される。
推薦した仲間にDMで渡してバトンを繋ぐ
04
ネットワーク図に緑ノードで追加される
推薦した人への矢印が伸び、チェーンが広がる。
多くの人から推薦されるほど矢印が集まり、信頼度が上がる
⚠ 招待コードのない方はどこからも登録できません。参加したい方はすでにネットワーク内にいる株クラ仲間にお声がけください。
