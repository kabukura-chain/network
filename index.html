// リアル人間株クラチェーン ネットワーク図

ピンチでズーム ／ ドラッグで移動
「株やってるやつでX（Twitter）やってないとかバーチャ！」
株クラ界隈ではX（Twitter）は必須ツール。でも初心者には誰をフォローすればいいかわからない。
BOT・詐欺師・情報商材屋・AIゾンビが山ほどいる中、 信頼できる仲間だけがリアル人間株クラチェーンとしてつながるネットワーク。本物の株クラアカウントを探せる・フォローできる唯一無二のサイト。誰が、本物の個人投資家なのか。
その残酷なまでの「答えあわせ」が、今ここから始まる。

// このネットワークの使い方　招待を受けた方は信頼できる3名の@Xハンドルを登録するのみ。

01 あなたの@ハンドルと、信頼できる株クラを最低1人〜最大3人入力して登録（1名でもOK）
02 ネットワーク図にあなたのブロックが追加され、推薦した人とつながって広がる
03 アイコンをタップするとXのアカウントページが開く。気になる人をすぐフォロー！
04 緑のノード＝登録者、青のノード＝推薦されたアカウント。同じ人が2回登録はできない
05 ノード数＝ネットワーク上に存在するアカウントの総数（登録者＋推薦されたアカウント）／コネクション＝アカウント同士のつながりの総数

📷 アイコンはX（Twitter）から自動取得されますが、アカウントの設定や状況により取得できない場合があります。その際は代替アイコンが表示されます。

⚠ 排除対象

BOT・詐欺師・情報商材屋・AIインプゾンビは紹介しないでください。このネットワークの価値は参加者の信頼で守られています。本物だと確信できる人だけを紹介してください。

// 不正アカウント通報フォーム

[ BOT・詐欺師・情報商材屋・AIゾンビを通報 ]

⚠ このネットワークの信頼を守るための通報フォームです。

  • 通報されたアカウントは通報状況に進捗が表示されます
  • 5名以上が同じアカウントを通報すると、自動的にネットワークから削除されます
  • 同じ人が同じアカウントを2回以上通報することはできません
  • @あり・なしどちらの入力でも登録されます

⚠ 身に覚えのない登録・削除申請

乗っ取り・なりすまし等により無断で登録された場合は、管理者までDMにてご連絡ください。ハンドル名と削除理由を添えてお送りいただければ対応します。
